# Contributing to Nuclear War Simulation

Thank you for your interest in contributing to this educational project! This simulation aims to promote awareness about nuclear weapons consequences and support nuclear disarmament efforts.

## 🎯 Project Mission

This project is designed for **educational purposes only** to:
- Increase awareness of nuclear weapons consequences
- Support nuclear disarmament advocacy
- Provide scientifically accurate educational content
- Promote peace through understanding

## 🤝 How to Contribute

### Reporting Issues
- Use the GitHub Issues tab to report bugs or suggest improvements
- Include detailed steps to reproduce any problems
- Provide information about your environment (OS, browser, etc.)

### Suggesting Enhancements
- Educational content improvements
- Additional cities or regions
- Better visualization features
- Accessibility improvements
- Translation support

### Code Contributions
- Fork the repository
- Create a feature branch from `develop`
- Make your changes with clear commit messages
- Submit a pull request to the `develop` branch

## 📋 Development Guidelines

### Code Standards
- Follow C# coding conventions
- Use meaningful variable and method names
- Add XML documentation for public methods
- Include unit tests for new features

### Educational Focus
- Maintain scientific accuracy
- Include appropriate disclaimers
- Focus on educational value
- Avoid glorifying violence

### Technical Requirements
- .NET 8 compatibility
- Responsive design (mobile-friendly)
- Accessibility compliance (WCAG 2.1)
- Cross-browser compatibility

## 🔬 Scientific Accuracy

When contributing calculations or data:
- Use peer-reviewed scientific sources
- Reference declassified government data when appropriate
- Include citations in code comments
- Validate formulas against published research

### Acceptable Sources
- "The Effects of Nuclear Weapons" (Glasstone & Dolan)
- Declassified nuclear testing data
- Peer-reviewed academic papers
- Historical records from Hiroshima and Nagasaki

## 🚫 Contribution Restrictions

We **DO NOT** accept contributions that:
- Provide weapons design information
- Include classified or sensitive data
- Glorify violence or warfare
- Promote harmful ideologies
- Compromise educational integrity

## 📝 Pull Request Process

1. **Fork** the repository
2. **Create** a feature branch: `git checkout -b feature/description`
3. **Commit** changes with descriptive messages
4. **Test** your changes thoroughly
5. **Submit** pull request to `develop` branch
6. **Respond** to code review feedback

### Pull Request Template
```
## Description
Brief description of changes

## Type of Change
- [ ] Bug fix
- [ ] New feature
- [ ] Educational content update
- [ ] Documentation improvement
- [ ] Performance enhancement

## Testing
- [ ] Local testing completed
- [ ] All existing tests pass
- [ ] New tests added (if applicable)

## Scientific Accuracy
- [ ] Sources verified and cited
- [ ] Calculations validated
- [ ] Educational value maintained
```

## 🌍 Internationalization

We welcome contributions for:
- Language translations

- Cultural sensitivity improvements
- Local educational context

All contributions must comply with the educational use license. By contributing, you agree that your contributions will be used for educational purposes only.
